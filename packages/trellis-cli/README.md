# @trellis/cli

Workspace CLI for the Trellis v3 GTM path.

Primary commands:

```bash
trellis init <target>
trellis docs add <path>
trellis doctor
trellis smoke
trellis deploy
trellis connect attio
trellis connect agentmail
trellis connect firecrawl
```

Legacy composition commands remain only for migration and require explicit legacy intent.
