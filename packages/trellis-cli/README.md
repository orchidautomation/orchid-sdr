# @trellis/cli

Workspace CLI for the Trellis GTM path.

Primary commands:

```bash
trellis init <target>
trellis docs add <path>
trellis doctor
trellis smoke
trellis deploy
trellis connect attio
trellis connect agentmail
trellis connect firecrawl
```

Unsupported composition commands are kept out of the default CLI surface.
